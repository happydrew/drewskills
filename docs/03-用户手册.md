# 用户手册

安装源：

```text
https://github.com/happydrew/drewskills.git
```

## 1. 安装整个仓库

当前仓库包含两个技能：

- `git-commit-and-push`
- `import-third-party-skill`

Claude Code：

```bash
npx skills add https://github.com/happydrew/drewskills.git --agent claude-code -g --yes
```

Codex：

```bash
npx skills add https://github.com/happydrew/drewskills.git --agent codex -g --yes
```

通用安装器：

```bash
npx openskills install https://github.com/happydrew/drewskills.git --universal -y
```

## 2. 安装单个技能

Claude Code / Codex：

```bash
npx skills add https://github.com/happydrew/drewskills.git --agent codex -g --skill import-third-party-skill --yes
```

通用安装器如需只装一个技能，应先获取仓库内容，再安装对应 skill 目录：

```bash
git clone https://github.com/happydrew/drewskills.git
npx openskills install ./drewskills/skills/import-third-party-skill --universal -y
```

## 3. 如何确认安装成功

- Claude Code：重启后确认技能可被发现和调用
- Codex：重启后执行 `/skills`
- 其他 agent：重启后确认新技能已可见

## 4. 如何贡献新技能

```powershell
python .\scripts\new_skill.py `
  --name my-new-skill `
  --title "My New Skill" `
  --description "One-line description"
python .\scripts\validate_registry.py
```

# 技能开发与贡献手册

## 1. 新技能位置

统一放在：

```text
skills/<skill-name>/
```

## 2. 创建方式

```powershell
python .\scripts\new_skill.py `
  --name my-new-skill `
  --title "My New Skill" `
  --description "One-line description" `
  --resources scripts,references
```

## 3. 约束

- 仓库级 `scripts/` 只放通用脚本
- 技能专属脚本、参考资料、示例必须放在技能目录下
- 不要把本地绝对路径写入 `SKILL.md`、示例命令或资源文件
- 不要让技能依赖作者本机目录结构

## 4. 提交前操作

```powershell
python .\scripts\validate_registry.py
```

## 5. 提交流程

```powershell
git add .
git commit -m "feat: update skills"
git push origin main
```

如果只是调整文档或仓库说明，也可以使用更合适的 `docs:` 或 `refactor:` 前缀。

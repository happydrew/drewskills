# 安装兼容说明

## 1. 当前安装设计

仓库按单一 skills source 维护：

- `skills/`
- `packs/`
- `hub.json`
- `AGENTS.md`

## 2. 当前支持的安装方式

- Claude Code：`npx skills add <repo-url> --agent claude-code -g --yes`
- Codex：`npx skills add <repo-url> --agent codex -g --yes`
- 通用 agent：`npx openskills install <repo-url> --universal -y`

单技能安装：

- Claude Code / Codex：`npx skills add <repo-url> --agent codex -g --skill <skill-name> --yes`
- 通用 agent：先获取仓库内容，再安装 `skills/<skill-name>`

## 3. 文档要求

- 不写本地绝对路径
- 不依赖当前机器目录结构
- 对外只保留可执行命令和必要说明
